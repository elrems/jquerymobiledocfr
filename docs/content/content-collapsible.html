Pour créer un bloc de contenu pliable, créez un conteneur et ajouter l'attribut data-role="collapsible". L'utilisation de l'attribut data-content-theme vous permet de définir un thème pour le contenu du pliable. Regardez la reference de data- attribute pour voir tous les attributs possibles que vous pouvez ajouter aux pliables.

Directement à l'intérieur de ce conteneur, ajoutez un élément entête (H1-H6) ou un élément légende. Le framework stylisera l'entête pour ressembler à un bouton cliquable et ajoutera une icône "+" à gauche pour indiquer qu'il est extensible.

Après l'entête, ajoutez n'importe quel balisage HTML que vous voulez être pliable. Le framework terminera ce balisage dans un conteneur qui sera caché/affiché lorsque l'entête sera cliqué.

Par défaut, le contenu sera réduit.

		
<div data-role="collapsible">
   <h3>Je suis l'entête</h3>
   <p>Je suis le contenu pliable. Par défaut je suis fermé, mais vous pouvez cliquer sur l'entête pour m'ouvrir. </p></div>

Je suis l'entête

Je suis le contenu pliable. Par défaut je suis fermé, mais vous pouvez cliquer sur l'entête pour m'ouvrir.

Déplier au chargement

Pour déplier le contenu quand la page se charge, ajoutez l'attribut data-collapsed="false" à l'enveloppe.

<div data-role="collapsible" data-collapsed="false">

Ce code va créer un widget pliable comme ceci :

Je suis l'entête

Je suis le contenu pliable. Je suis déplier par défaut, parce que j'ai l'état "collapsed" défini à false.

Pliables Non intégrés

Par défaut les pliables ont une apparence intégré. Pour les rendre sur toute la largeur sans coins, ajoutez l'attribut data-inset="false" à l'élément.

<div data-role="collapsible" data-inset="false">

Ce code créera un pliable non-intégré :

Je suis le contenu pliable.

Mini pliables

Pour avoir une version plus compacte, qui est utile dans les barres d'outils et les espaces restreints, ajoutez l'attribut data-mini="true" à l'élément pour créer une version mini.

<div data-role="collapsible" data-mini="true">

Ce code va créer un mini widget pliable comme ceci :

Icônes personnalisées

Le positionnement par défaut des icônes dans les entêtes des pliables peut être modifié en utilisant les attributs data-collapsed-icon et data-expanded-icon. Dans l'exemple ci-dessous, nous avonsdata-collapsed-icon="arrow-r" et data-expanded-icon="arrow-d".

J'ai des icônes personnalisées

data-collapsed-icon="arrow-r" et data-expanded-icon="arrow-d"

Positionnement d'icône

Le positionnement par défaut des icônes dans les entêtes des pliables peut être modifié en utilisant l'attribut data-iconpos. Dans le cas ci-dessous, nous avons data-iconpos="right".

Je suis l'entête

data-iconpos="right"

Thématisation du contenu pliable

Le contenu pliable est peu stylisé - nous ajoutons seulement un peu de marge entre la barre et le contenu, l'entête adopte les styles du thème par défaut du conteneur où il se trouve.

Pour fournir une connexion visuelle plus forte entre l'entête pliable et le contenu, ajouter l'attribut data-content-theme à l'enveloppe et spécifiez une lettre de nuancier de thème. Ceci applique aux bordures du nuancier et à la couleur de fond flat (pas le gradient) pour le bloc de contenu et modifiera l'arrondi des coins pour du bas de l'entête et arrondira le bas du bloc de contenu afin de grouper visuellement ces éléments.

		
<div data-role="collapsible" data-content-theme="c">
   <h3>Entête</h3>
   <p> Je suis le contenu pliable avec un bloc de contenu thématique à "c". </p></div>

Entête

Je suis le contenu pliable avec un bloc de contenu thématique à "c".

Thématisation des entêtes pliables

Pour définir le thème sur un bouton d'entête pliable, ajoutez l'attribut data-theme à l'enveloppe et spécifiez une lettre de nuancier. Notez que vous pouvez mélanger et assortir les lettres de nuancier entre l'entête et le contenu avec ces attributs de thème.

		
<div data-role="collapsible" data-theme="a" data-content-theme="a">
   <h3>Entête nuancier A</h3>
   <p> Je suis le contenu pliable avec un bloc de contenu thématique à "a". </p></div>

Entête nuancier A

Je suis le contenu pliable avec un bloc de contenu thématique à "a".

Entête nuancier B

Je suis le contenu pliable avec un bloc de contenu thématique à "d".

Styliser les coins

Pour créer des pliables intégrés sans coins arrondis, vous pouvez ajouter l'attribut data-corners="false" à l'élément.

<div data-role="collapsible" data-corners="false" data-theme="b" data-content-theme="d">

Le code créera un pliable intégré sans les coins arrondis :
