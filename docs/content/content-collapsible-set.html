Ensemble de pliable (accordéon)

  • Bases
  • Options
  • Méthodes
  • Événements

Les ensembles pliable commencent avec exactement le même balisage que les pliables individuels. En ajoutant une enveloppe parent avec l'attribut data-role="collapsible-set" autour d'un certain nombre de pliables, le framework le stylisera pour ressemble à un widget visuel groupé et le fera se comporter comme un accordéon dont une seule section peut être ouvert à la fois. Regardez la reference de data- attribute pour voir tous les attributs possibles que vous pouvez ajouter aux ensembles de pliable.

Par défaut, toutes les sections seront réduites. Pour définir une section à ouvrir lorsque la page se charge, ajoutez l'attribut data-collapsed="false" à l'entête de la section que vous souhaitez dépliée.

		
<div data-role="collapsible-set">

	<div data-role="collapsible" data-collapsed="false">
	<h3>Section 1</h3>
	<p>Je suis le contenu pliable pour la section 1.</p>
	</div>
	
	<div data-role="collapsible">
	<h3>Section 2</h3>
	<p>Je suis le contenu pliable pour la section 2.</p>
	</div>
	
</div>

Thématisation du contenu pliable

L'attribut standard data-theme peut être utilisé pour définir la couleur de chaque pliable dans un ensemble. Pour fournir un regroupement plus claire visuellement du contenu des entêtes, ajoutez l'attribut data-content-theme avec une lettre de nuancier. Cela ajoute une couleur de fond thématisé et des bordures pour le bloc de contenu. Pour une thématisation cohérente, ajouter ces attributs à l'ensemble pliable parent .

		
<div data-role="collapsible-set" data-theme="c" data-content-theme="d">

Styliser les coins

Ajoutez l'attribut data-corners="false" pour avoir un ensemble de pliable intégré sans les coins arrondis.

		
<div data-role="collapsible-set" data-corners="false" data-theme="b" data-content-theme="d">
