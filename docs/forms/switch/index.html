Un interrupteur "flip" binaire est un élément d'interface commune sur les appareils mobiles qui est utilisé pour saisie de données binaires marche/arrêt ou vrai/faux . Vous pouvez soit faire glisser l'interrupteur comme un curseur ou appuyez sur un des côtés de l'interrupteur.

Pour créer un interrupteur, commencer avec un select avec l'attribut data-role="slider" et deux options. La première option sera stylisé comme un interrupteur à l'état "marche" et le second sera stylisé comme un interrupteur à l'état "arrêt" de ce fait, écrivez vos options en conséquence. Regardez la référence de l'attribut data- pour voir tous les attributs possibles que vous pouvez ajouter aux interrupteurs.

Réglez l'attribut for de label pour faire correspondre l'id de l'input ainsi ils sont sémantiquement liés. Il est possible de cacher le label d'accessibilité si il n'est pas souhaitée dans la mise en page, mais nous avons besoin qu'il soit présent dans le balisage pour des raisons de sémantique et d'accessibilité.

	
<label for="flip-1">Interrupteur :</label>
<select name="flip-1" id="flip-1" data-role="slider">
	<option value="off">Arrêt</option>
	<option value="on">Marche</option>
</select>

Ceci va produire un interrupteur. Les styles par défaut définissent la largeur de l'interrupteur à 100% du conteneur parent et empile les labels sur une ligne distincte.

Labels plus long

Le contrôle est proportionnellement réduite, afin d'utiliser des labels plus longs, on peut simplement ajouter une ligne de CSS mettant l'interrupteur à la largeur désirée. Par exemple, voici le balisage suivant :


<div class="containing-element">
	<label for="flip-min">Interrupteur :</label>
	<select name="flip-min" id="flip-min" data-role="slider">
		<option value="off">Interrupteur Off</option>
		<option value="on">Interrupteur On</option>
	</select>
</div>

Comme certains styles par défaut dépendent des fieldcontains, notez que vous devrez peut-être veiller à ce que des styles personnalisés s'applique aux interrupteurs au sein des fieldcontains en utilisant .ui-field-contain div.ui-slider-switch { width: [...]; }.

Version mini

Pour avoir une version plus compacte, qui est utile dans les barres d'outils et les espaces restreints, ajoutez l'attribut data-mini="true" à l'élément pour créer une version mini. 

	
<label for="flip-mini">Interrupteur :</label>
<select name="flip-mini" id="flip-mini" data-role="slider" data-mini="true">
	<option value="off">Arrêt</option>
	<option value="on">Marche</option>
</select>

Ceci va produire un interrupteur pas aussi grand que la version standard et disposant d'une taille inférieure.

Conteneurs de champs

Optionnellement, enveloppez le balisage de l'interrupteur dans un conteneur avec l'attribut data-role="fieldcontain" afin de les grouper visuellement dans un formulaire plus long.

	
<div data-role="fieldcontain">
<label for="flip-2">Interrupteur :</label>
	<select name="flip-2" id="flip-2" data-role="slider">
		<option value="nope">Non</option>
		<option value="yep">Oui</option>
	</select> 
</div>

Thématisation de l'interrupteur

Comme tous les éléments de formulaire, ce widget héritera automatiquement du thème du conteneur parent. Pour choisir un nuancier de couleur de thème spécifique, spécifiez l'attribut data-theme sur le select et spécifiez une lettre de nuancier.

	
<div data-role="fieldcontain">
	<label for="flip-3">Interrupteur :</label>
	<select name="flip-3" id="flip-3" data-role="slider" data-theme="a">
		<option value="no">Non</option>
		<option value="yes">Oui</option>
	</select> 
</div>

Il en résulte un interrupteur avec les couleurs du nuancier "a" pour le curseur. Notez que le statut à gauche "Marche" obtient la couleur du statut actif.

Il est également possible de thématiser la piste de l'interrupteur en ajoutant l'attribut data-track-theme et en spécifiant la lettre du nuancier choisi sur le select.

Voici un exemple d'un interrupteur avec le nuancier "a" appliqué à la piste et le nuancier "c" appliqué à la poignée :

	
<div data-role="fieldcontain">
        <label for="flip-5">Interrupteur :</label>
        <select name="flip-5" id="flip-5" data-role="slider" data-theme="c" data-track-theme="a">
                <option value="no">Non</option>
                <option value="yes">Oui</option>
        </select>
</div>

Appel du plugin interrupteur

Ce plugin sera auto-initialiser sur n'importe quelle page qui contient un select avec l'attribut data-role="slider". Cependant, si nécessaire, vous pouvez appeler directement le plugin curseur sur un sélecteur,comme n'importe quel plugin jQuery :


$('select').slider();
