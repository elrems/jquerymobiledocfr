Thémes de formulaires

jQuery Mobile possède un riche système de thématisation qui vous donne le contrôle totale pour que les pages et les formulaires soient stylisés. Par défaut, tous les éléments de formulaire dans un conteneur vont automatiquement prendre le nuancier de couleur du thème de leur parent. Cela permet aux éléments de formulaire de se fondre dans leur mise en page avec un minimum de travail. L'attribut data-theme peut être appliqué individuellement à n'importe quel élément du formulaire pour appliquer l'une des lettres des nuanciers de couleur du thème pour créer un contraste et accent dans vos conceptions.

Tous les éléments du formulaire dans les exemples ci-dessous utilisent le même code HTML sans nuancier de thème spécifié sur les éléments du formulaire. La seule différence entre chaque bloc de code est un nuancier de couleur data-themeattribué à chaque conteneur parent.
