Liez les événements directement à l'élément input. Utilisez les évènements virtuels de jQuery Mobile ou liez les évènements standard de JavaScript tel que change, focus, blur, etc... :

 
$("input[type='checkbox']").on( "change", function(event, ui) {
  ...
});

Le plugin case à cocher a les événements personnalisés suivants :

create déclenché quand une case est créée

$("input[type='checkbox']").checkboxradio({
   create: function(event, ui) { ... }
});
