Débuter avec jQuery Mobile

jQuery Mobile fournit un ensemble de widgets d'interface utilisateur tactile conviviale et un système de navigation AJAX pour supporter les transitions animées des pages. Construire votre première page jQuery Mobile c'est facile. Voici comment faire :

Créer un modèle de page de base

Ouvrez votre éditeur de texte favori et collez ce qui suit dans le modèle de page, sauvez et ouvrez le dans un navigateur. Vous êtes maintenant un développeur mobile !

Voici ce qu'il y a dans le modèle. Dans le head, une balise meta : viewport, définit la largeur de l'écran à celle de la largeur en pixel de l'appareil, de plus des références à jQuery, à jQuery Mobile, à la feuille de style du thème du mobile depuis le CDN, et tous les styles et les scripts. jQuery Mobile 1.3 beta (1.3.0-beta.1) fonctionne avec les versions jQuery de la 1.7.0 à la 1.8.3.

Dans le body, un div avec un data-role à page est l'enveloppe utilisée pour délimiter une page, la barre d'entête (data-role="header") et la région du contenu (data-role="content") sont ajoutés à l'intérieur pour créer une page de base (ceux-ci sont tous les deux facultatifs). Ces attributs data- sont des attributs HTML5 qui sont utilisés à travers jQuery Mobile pour transformer un balisage de base en un widget amélioré et stylisé.


<!DOCTYPE html> 
<html> 
<head> 
	<title>My Page</title> 
	<meta name="viewport" content="width=device-width, initial-scale=1"> 
	<link rel="stylesheet" href="http://code.jquery.com/mobile/1.3.0-beta.1/jquery.mobile-1.3.0-beta.1.min.css" />
	<script src="http://code.jquery.com/jquery-1.8.3.min.js"></script>
	<script src="http://code.jquery.com/mobile/1.3.0-beta.1/jquery.mobile-1.3.0-beta.1.min.js"></script>
</head> 
<body> 

<div data-role="page">

	<div data-role="header">
		<h1>Mon titre</h1>
	</div><!-- /header -->

	<div data-role="content">	
		<p>Hello world</p>		
	</div><!-- /content -->

</div><!-- /page -->

</body>
</html>

Ajoutez votre contenu

À l'intérieur de votre conteneur de contenu, vous pouvez ajouter les éléments HTML standard - titres, listes, paragraphes, etc. Vous pouvez écrire vos propres styles personnalisés pour créer des mises en page personnalisées en ajoutant une feuille de style au head après la feuille de style de jQuery Mobile.

Faire une liste

jQuery Mobile comprend un ensemble diversifié de listes communes qui sont codées comme des listes avec l'ajout d'un data-role="listview". Voici une simple liste qui a un rôle listview. Nous allons changer l'apparence pour que cela ressemble à un seul bloc en ajoutant un data-inset="true" et nous ajouterons un filtre de recherche dynamique avec l'attribut data-filter="true".


<ul data-role="listview" data-inset="true" data-filter="true">
	<li><a href="#">Acura</a></li>
	<li><a href="#">Audi</a></li>
	<li><a href="#">BMW</a></li>
	<li><a href="#">Cadillac</a></li>
	<li><a href="#">Ferrari</a></li>
</ul>
  • Acura
  • Audi
  • BMW
  • Cadillac
  • Ferrari

Ajouter un curseur (slider)

Le framework contient un ensemble complet d'éléments de formulaire qui sont automatiquement améliorés en widgets tactiles conviviales et stylisés. Voici un curseur qui est fait avec un nouvel input HTML5 de type range, il n'y a pas besoin de data-role. Assurez vous de l'envelopper dans un élément form et de toujours bien associer un label pour chaque élément du formulaire.


<form>
   <label for="slider-0">Curseur :</label>
   <input type="range" name="slider" id="slider-0" value="25" min="0" max="100"  />
</form>

Faire un bouton

Il existe plusieurs façons de faire des boutons, mais la transformation d'un lien en bouton est plus facile à cliquer. Il suffit de commencer avec un lien et de lui ajouter un attribut data-role="button". Vous pouvez ajouter une icône avec l'attribut data-icon et éventuellement définir sa position avec l'attribut data-iconpos.


<a href="#" data-role="button" data-icon="star">Bouton étoile</a>
Bouton étoile

Jouer avec les nuanciers du thème

jQuery Mobile a un framework de thème solide qui supporte jusqu'à 26 ensembles de barre d'outils, de contenu et de bouton, appelé "swatch" (Nuancier). Ajoutez juste l'attribut data-theme="e" à n'importe quels widgets sur cette page : page, entête, liste, curseur ou bouton pour le passer en jaune. Essayez les différentes lettres de nuancier (de a à e) dans le thème par défaut pour mélanger et assortir les nuanciers.

Astuce intéressante : ajoutez le nuancier du thème à la page et regardez comment tous les widgets se trouvant à l'intérieur du contenu hériteront automatiquement du thème (les entêtes et les pieds de pages n'héritent pas, ils sont par défaut avec le nuancier "a").

<a href="#" data-role="button" data-icon="star" data-theme="a">Button</a> data-theme="a" data-theme="b" data-theme="c" data-theme="d" data-theme="e"

Lorsque vous êtes prêt à créer un thème personnalisé, utilisez ThemeRoller, puis télécharger le thème personnalisé.

Pour continuer et construire des choses

Ce n'est qu'effleurer la surface de toutes les choses cool que vous pouvez construire sans trop d'effort avec jQuery Mobile. N'oubliez pas d'explorer la liaison de pages, l'ajout des transitions animées des pages et la création des boîtes de dialogue. Utilisez la référence des attributs data- afin de pouvoir jouer avec d'autres attributs data-.

Plus d'infos pour le développeur ? Bien, oubliez tout ce que nous venons de voir (blague). Si vous ne voulez pas utiliser le système de configuration de l'attribut data-, vous pouvez prendre le contrôle total de tout et appelez les plugins directement, parce que ce sont simplement tous des plugins jQuery standards intégrés avec l'ensemble des widgets de l'interface utilisateur. Veillez à creuser dans la configuration globale, les événements et les méthodes.
