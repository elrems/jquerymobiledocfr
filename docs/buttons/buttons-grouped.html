Groupement des boutons

Occasionnellement, vous voudrez peut-être un ensemble de boutons pour former un seul bloc qui ressemble à une composante de navigation. Pour obtenir cet effet, envelopper un ensemble de boutons dans un container avec l'attribut data-role="controlgroup" — le framework créera un groupe de boutons verticale, enlèvera toutes les marges et les ombres portées entre les boutons et laissera seulement l'arrondi du premier et dernier bouton de la série pour créer un effet de regroupement. 


<div data-role="controlgroup">
<a href="index.html" data-role="button">Oui</a>
<a href="index.html" data-role="button">Non</a>
<a href="index.html" data-role="button">Peut être</a>
</div>

By default, grouped buttons are presented as a vertical list:

Oui Non Peut être

En ajoutant l'attribut data-type="horizontal" au container controlgroup, vous pouvez transformer en un groupe horizontal qui positionne les boutons côte à côte et définir une largeur pour être assez grand pour accueillir le contenu. (Soyez conscient que cela va se positionner sur plusieurs lignes si le nombre de boutons ou la longueur du texte global est trop large pour l'écran.)
