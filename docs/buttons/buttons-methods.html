Les méthodes suivantes s'appliquent uniquement aux boutons de formulaire. Les boutons basé sur un lien basé n'ont pas de méthodes associées.

enable activer un bouton de formulaire désactivé

$('[type="submit"]').button('enable');
disable désactiver un bouton de formulaire

$('[type="submit"]').button('disable');
refresh mise à jour le bouton du formulaire

Si vous manipulez un bouton de formulaire via JavaScript, vous devez appeler la méthode refresh sur le bouton pour mettre à jour le style visuel.

		
$('[type="submit"]').button('refresh');
