Liez les événements directement à l'ensemble de conteneur, typiquement un élément div. Utilisez les évènements virtuels de jQuery Mobile ou liez les évènements standard de JavaScript tel que change, focus, blur, etc... :


$( ".selector" ).on( "create", function(event, ui) {
  ...
});

Le plugin ensemble de pliable a les événements personnalisés suivants :

create déclenché quand un ensemble de pliable est créé

$( ".selector" ).collapsibleset({
   create: function(event, ui) { ... }
});
