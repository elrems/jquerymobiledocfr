L'approche par défaut pour styliser le contenu in jQuery Mobile est simple : Utilisez une main légère. Notre objectif est de laisser le rendu natif du navigateur prioritaire, nous avons ajouter un peu d'espacement pour une lisibilité plus confortable et d'utiliser le système de thématisation pour appliquer les familles de polices et les couleurs.

Prendre une main légère avec un style de contenu permet aux concepteurs et aux développeurs de travailler avec une ardoise propre, plutôt que de lutter contre un grand nombre de style généraux complexes.

Balisage HTML par défaut du style
