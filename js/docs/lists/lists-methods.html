Le plugin listview a les méthodes suivantes :

childPages récupère les sous-pages

Cette méthode retourne un objet jQuery contenant toutes les pages enfants immédiats d'une liste imbriquée.


$('.selector').listview('childPages');
refresh met à jour la liste

Si vous manipulez une liste via JavaScript (par exemple vous ajoutez des éléments LI), vous devez appeler la méthode refresh sur elle même pour mettre à jour le style visuel.


$('.selector').listview('refresh');
