Navigation : Pages & Boîtes de dialogue

jQuery Mobile inclut un système de navigation pour charger les pages liées dans le DOM via AJAX, améliorer le nouveau contenu, puis afficher les pages avec un vaste éventail de transition pour animer les pages. Le système de navigation transforme automatiquement tous les liens et les formulaires à l'aide de l'amélioration progressive pour détourner les liens et émettre des requêtes AJAX.

Le bouton retour est entièrement pris en charge afin que les pages, les boîtes de dialogue et les popups fonctionnent parfaitement avec le système de navigation. Il existe des outils pour les pages pour précharger & mettre en cache, injecter dynamiquement et faire des scripts dans les cas d'utilisation avancés.
