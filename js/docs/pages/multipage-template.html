Un

J'ai un id "one" sur ma page conteneur. Je suis le premier dans l'ordre de la source, donc je suis affiché lorsque la page se charge.

C'est un modèle passe-partout multi-pages que vous pouvez copier pour construire votre première page jQuery Mobile. Ce modèle contient plusieurs conteneurs "page" à l'intérieur, contrairement à un modèle de page unique qui n'a qu'un "page" à l'intérieur.

Affichez la source et copiez le code pour démarrer. Toutes les CSS et les JS sont liés aux versions CDN de jQuery donc c'est super facile à mettre en place. N'oubliez pas d'inclure une balise META viewport dans le head pour régler le niveau du zoom.

Vous liez des pages internes en se référant à l'id de la page que vous voulez montrer. Par exemple, pour le lien à la page avec un id "two", mon lien aura un href="#two" dans le code.
