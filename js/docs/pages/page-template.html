Page unique

C'est un modèle passe-partout d'une page unique que vous pouvez copier pour construire votre première page jQuery Mobile. Chaque lien ou formulaire, à partir d'ici, tirera une nouvelle page via Ajax pour supporter les transitions de page animée.

Affichez la source et copiez le code pour démarrer. Toutes les CSS et les JS sont liés aux versions CDN de jQuery donc c'est super facile à mettre en place. N'oubliez pas d'inclure une balise META viewport dans le head pour régler le niveau du zoom.

Ce modèle est un document HTML standard avec un unique conteneur "page" à l'intérieur, contrairement à un modèle multi-page qui a plusieurs pages à l'intérieur. Nous recommandons fortement la construction de votre site ou de votre application comme une série de pages séparées parce que c'est plus propre, plus léger et fonctionne mieux sans JavaScript.
