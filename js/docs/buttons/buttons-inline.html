Alignement des boutons

Par défaut, tous les boutons dans le contenu du corps sont stylisés comme élément de bloc-niveau afin qu'ils remplissent la largeur de l'écran :

Bouton

Toutefois, si vous voulez un bouton plus compact qui soit seulement aussi large que le texte et les icônes à l'intérieur, ajoutez l'attribut data-inline="true" au bouton :

Bouton

Si vous avez plusieurs boutons qui doivent se positionner côte à côte sur la même ligne, ajoutez l'attribut data-inline="true" à chaque bouton. Cela stylisera les boutons avec la largeur de leur contenu et alignera les boutons afin qu'ils soient positionner sur la même ligne. 


<a href="index.html" data-role="button" data-inline="true">Annuler</a>
<a href="index.html" data-role="button" data-inline="true" data-theme="b">Sauver</a>

Le résultat est le suivant :

Annuler Sauver

En ajoutant data-mini="true" aux boutons alignés, cela créé une version plus compacte :

Annuler Sauver

Si vous voulez des boutons positionnés côte à côte, mais étirés pour remplir la largeur de l'écran, vous pouvez utiliser les grilles de de colonne de contenu pour utiliser toute la largeur sur 2 ou 3 colonnes.
